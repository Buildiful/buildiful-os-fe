# App Packs

BOS is designed to support app-pack style expansion.

## Intended model

App packs live under:

```text
~/Buildiful/apps/<pack-slug>/
```

Each pack is expected to include at minimum:

- `manifest.env`
- `README.md`

## Founder Edition status

Founder Edition does not yet ship a full pack library.

Instead, it provides the base workspace and CLI needed for future packs and accompanying ebooks.

## Likely examples

Future pack categories may include:

- trading agent
- CD laddering agent
- home vault agent
- other local-first agent workflows
