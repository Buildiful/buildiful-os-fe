# BOS Commands

## `bos version`

Shows BOS name, version, channel, and base OS.

## `bos status`

Shows:

- machine info
- install state
- machine profile status
- core workspace directory status
- tooling availability
- installed packs

## `bos doctor`

Runs health checks and reports:

- PASS
- WARN
- FAIL

## `bos dirs`

Explains the Buildiful workspace folders.

## `bos apps`

Lists installed app packs and manifest info when available.

## `bos docs`

Shows where local docs are installed.

## `bos backup`

Creates a timestamped backup archive of available Buildiful config, data, logs, and apps.

## `bos launch`

Opens the BOS home screen or local docs when available.
