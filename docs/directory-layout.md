# Directory Layout

BOS creates this workspace under your home directory:

```text
~/Buildiful/
```

## Folders

- `apps/` installed app packs and local app bundles
- `agents/` runtime working files for agents
- `config/` BOS config, templates, machine profile, install state
- `data/` structured local-first data
- `docs/` local BOS docs and examples
- `logs/` install and runtime logs
- `backups/` generated backup archives
- `scripts/` helper scripts
- `models/` local model storage or references
- `inbox/` drop zone for files to be processed
- `exports/` reports and generated outputs
- `branding/` release notes and local branding assets
