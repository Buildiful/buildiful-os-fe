# Getting Started with Buildiful OS [Founder Edition]

Buildiful OS [Founder Edition] is a free local-first workspace layer for Ubuntu 24.04.

It helps turn your own computer into a structured AI agent workstation by setting up:

- a Buildiful workspace under your home directory
- a BOS command surface
- baseline local tooling
- machine metadata and install state
- a foundation for future app packs

## First commands to run

After installation:

```bash
bos version
bos status
bos doctor
bos dirs
bos docs
```

## Buildiful workspace

BOS uses this root directory:

```text
~/Buildiful
```

Inside it, BOS creates folders for apps, agents, config, data, docs, logs, backups, and more.

## Current scope

Founder Edition is intentionally lightweight.

It is a base layer, not a full app marketplace or a polished commercial automation suite.

## Next direction

BOS is meant to pair with Buildiful ebooks and future local-first agent packs.
